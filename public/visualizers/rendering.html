What Happens During Rendering

Render Phase (Interruptible)

  • • Component functions are called
  • • Hooks execute (useState, useReducer, etc.)
  • • JSX → React.createElement()
  • • React Elements created (Virtual DOM)
  • • Reconciliation (diffing algorithm)
  • • Fiber flags marked (Placement, Update, Deletion)
  • • Work-In-Progress tree built

Commit Phase (Synchronous)

  • • DOM mutations applied
  • • Ref updates attached
  • • useLayoutEffect (synchronous)
  • • Browser paint
  • • useEffect (asynchronous, after paint)
